Made purely for fun and to learn more about C. fully inspired from [Daniel Hirsch](https://www.youtube.com/watch?v=sItRLFjbqvo) He's way better than me at this!
# Why I made it
I was bored on a casual Tuesday and came across a video from Daniel and decided to follow his footsteps.
# Installation (if you wanna test it out)
It is super unreliable so don't complain too much
```bash
git clone ...
cd ImageViewer
make
# in the Makefile, change the "filename" variable to your file directory. PPM only. Import in binary from GIMP not ASCII.
make r # see your picture rendered! 
```
